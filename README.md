## What is SheBuilds x CentraleTech?

SheBuilds is a **global build day** organized by [Lovable](https://lovable.dev) to celebrate International Women's Day.

CentraleTech is hosting a **local node** at École Centrale Casablanca: a focused build sprint where everyone — regardless of coding level — ships a **real, functional, and impactful product** using Lovable's AI-powered platform.

> No excuses. No "just an idea". Ship it.

---

## The Challenge

Build something that is:

| Criteria | Description |
|---|---|
| ✅ **Functional** | It works. For real. Not a mockup. |
| 💡 **Useful & Impactful** | It solves a real problem for real people. |
| 🎨 **Well Designed** | Clean UI, great UX, accessible. |
| 📐 **Ergonomic** | Intuitive to use, no friction. |
| 📈 **Scalable** | Built to grow, not just for the demo. |
| ⚡ **Efficient** | Fast, lightweight, no bloat. |

## How to Submit Your Project

1. **Build** your project on [Lovable](https://lovable.dev/invite/I5Q4L6B) during the build day.
2. **Publish** it (Lovable gives you a live URL instantly).
3. **Fork** this repo.
4. **Create a folder** under `projects/` named after your project:
 ```
 projects/your-project-name/
 ├── README.md ← description, problem solved, demo link, stack
 └── demo.gif or screenshot.png ← visual of your app
 ```
5. **Open a Pull Request** with the title: `[PROJECT] Your Project Name`.

### Your project README.md must include:

```markdown
## Project Name
**Team:** [Names]
**Demo:** [Live Lovable URL]
**Problem:** What problem does this solve?
**Solution:** How does your app solve it?
**Stack:** Lovable + [anything else]
**Screenshots / Video:** [link or embedded]
```
