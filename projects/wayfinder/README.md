Wayfinder

**Team:** Mamadou Berthé
**Demo:** https://guide-professionnel.lovable.app
**Problem:** 60% of African graduates are unemployed despite having diplomas. They lack guidance on how to become employable, start a business, or choose the right career path.
**Solution:** Wayfinder offers 3 AI-powered paths: Find a Job, Launch a Business, and Career Orientation — personalized by country, field, and budget for all 54 African countries. Available in French, English and Arabic.
**Stack:** Lovable + Anthropic Claude API + Pexels API
